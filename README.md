# FoodGallery-Using-React
Food Gallery is a simple React application that displays different food items in a gallery format. The application uses React components, arrays, and the map() function to dynamically render food images and their names. Each food item is displayed inside a separate card, creating an attractive and organized gallery layout.

## Features
* Displays multiple food items with images.
* Uses React functional components.
* Dynamic rendering using the map() method.
* Responsive gallery layout with CSS styling.
* Simple and user-friendly interface.

## Technologies Used
* React JS
* JavaScript
* HTML
* CSS

## Food Items Displayed
* Pizza
* Burger
* Biryani
* Dosa
* Cake
* Ice Cream
* Sandwich
* French Fries

## Conclusion
This project demonstrates the basic concepts of React such as components, JSX, image imports, arrays, and dynamic rendering using the map() function. It provides a visually appealing food gallery and helps beginners understand React fundamentals.
